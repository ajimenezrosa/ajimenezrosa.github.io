# Introducción

Este documento constituye mi plan maestro de formación profesional entre
2026 y 2029. Su propósito es consolidar una carrera como **Database
Infrastructure Engineer**, combinando administración avanzada de bases
de datos, sistemas Linux Enterprise, automatización e infraestructura
moderna.

Los recursos seleccionados priorizan cursos oficiales, certificaciones
reconocidas por la industria y formación práctica.

------------------------------------------------------------------------

# Objetivos

-   Dominar Linux Enterprise (Red Hat/Fedora).
-   Convertirme en especialista en SQL Server, PostgreSQL, Oracle y
    MongoDB.
-   Automatizar tareas con Bash, Python y Ansible.
-   Obtener certificaciones internacionales.
-   Desarrollar laboratorios y proyectos reales.

------------------------------------------------------------------------

# Perfil Profesional Objetivo

Al finalizar esta ruta deberé estar preparado para desempeñarme como:

-   Senior Database Administrator
-   Linux Systems Engineer
-   Database Infrastructure Engineer
-   Platform Engineer
-   Database Architect

------------------------------------------------------------------------

# Filosofía de Estudio

-   Estudiar entre 10 y 15 horas por semana.
-   Priorizar la práctica sobre la teoría.
-   No presentar una certificación sin haber completado laboratorios.
-   Documentar todo el aprendizaje en GitHub.

**Duración estimada:** 12 semanas

-   Permisos
-   SELinux
-   systemd
-   Networking
-   Firewalld
-   LVM

-   SSH

    https://www.udemy.com/courses/search/?q=linux+administration+bootcamp
