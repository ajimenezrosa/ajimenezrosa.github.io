# FASE 6 -- SQL Server (DBA Senior)
### [***Realizado y Documentado en paginas***]

## Objetivo

Dominar la administración de SQL Server en entornos empresariales de
alta disponibilidad y rendimiento.

**Prioridad:** ⭐⭐⭐⭐⭐\
**Duración:** 16 semanas

## Competencias

-   Instalación y configuración
-   Seguridad
-   Backup y Restore
-   AlwaysOn Availability Groups
-   Log Shipping
-   Replicación
-   Query Store
-   Wait Statistics
-   TempDB
-   Índices
-   Columnstore
-   Performance Tuning
-   Troubleshooting

## Cursos recomendados

### Microsoft Learn

https://learn.microsoft.com/training/

### Udemy (buscar)

-   SQL Server Administration
-   SQL Server Performance Tuning
-   SQL Server Internals
-   SQL Server AlwaysOn

## Certificación

Azure Database Administrator Associate

https://learn.microsoft.com/credentials/certifications/azure-database-administrator-associate/

**Proyecto recomendado**

Construir un laboratorio con dos instancias SQL Server y AlwaysOn sobre
Linux o Windows.
